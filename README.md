# UniGuide - 以高中生為客群的分科升學導航 APP

UniGuide 是一款整合科系歷屆錄取分數、熱門程度、落點分析、升學倒數日等多項功能，提供分科測驗「一站式服務」的 iOS 應用程式。本系統專為台灣升學體制設計，提供直覺的使用者介面與 iOS 桌面小工具，讓考生能隨時掌握升學資訊與重要日程。

## 核心功能

* 大學落點預測：基於 112 至 114 學年度的大考累積人數與科系標準，提供詳細的錄取率預測與分析。
* 成績換算系統：支援級分與百分位數的相互轉換，自動解析各科權重與超額比序。
* 日程管理與追蹤：內建個人化日程表，並支援 iOS 桌面小工具 (Widgets)，於設備主畫面便捷查看重要日程。
* 校系探索與收藏：提供各大專院校與科系的詳細資訊查詢、排名比較及個人書籤功能。
* 預測歷史追蹤：自動儲存使用者的落點分析紀錄，方便日後覆盤與志願調整。
* 帳號與資料同步：整合使用者認證服務，安全儲存並管理個人成績、收藏與日程資料。

## 技術架構

* 開發語言：Swift 5.0+
* 應用程式框架：主要基於 SwiftUI 構建，部分過渡橋接套用 UIKit。
* 擴充功能：使用 WidgetKit 開發 ScheduleWidget 桌面小工具。
* 第三方服務整合：
  * Google 服務 / Firebase (內含身份驗證等服務整合)。
  * 應用程式內廣告服務 (實作了開屏廣告與獎勵式廣告管理)。
* 數據庫與資料解析：將龐雜的歷年考情轉換為 CSV 格式於本地儲存，提供快速的離線與本地解析算法。

## 專案結構簡介

主要區分為核心應用層與小工具擴充層：

* `UniGuide/`：主應用程式邏輯與 UI 實作。
  * `Models/`：系統資料模型，包含預測成果、日程、使用者等資料結構。
  * `Methods/`：分析演算法核心，負責落點預測、成績換算、以及各類 CSV 資料表解析。
  * `Services/`：管理外部服務與應用狀態，涵蓋認證、廣告控制、歷史紀錄管理與計時器服務。
  * `Views/`：SwiftUI 視圖組件庫，包含主標籤頁、選單、以及各式互動畫面。
  * `Resources/`：歷年學測與分發的靜態 CSV 數據檔案庫及字體等資源。
  * `Extensions/`：各類原生型別 (Color、String 等) 的擴充方法。
* `ScheduleWidget/`：iOS 桌面小工具模組，獨立實作介面邏輯並共享核心日程數據。

## 系統需求

* 開發環境：Xcode 15.0 或以上版本建議。
* 執行環境：iOS 16.0 或以上版本 (需支援 SwiftUI 現代特性與 WidgetKit)。
* 第三方依賴：依賴相關第三方 SDK，由 Swift Package Manager (或 CocoaPods) 進行管理。

## 安裝與執行指引

1. 取得專案：將本專案原始碼同步拉取至本地開發環境。
2. 憑證與防護設定：請確保 `UniGuide/GoogleService-Info.plist` 已正確放置於專案根目錄中，以維持 Firebase 等相關服務正常運作。
3. 開啟專案：請使用 Xcode 開啟 `UniGuide.xcodeproj` (或 `UniGuide.xcworkspace`)。
4. 依賴解析：開啟專案後，請等待 Xcode 自動解析並下載所需的外部依賴套件。
5. 建置與執行：選擇合適的模擬器或連線的 iOS 實體測試機，點擊 Run 進行編譯與應用程式測試。

## 注意事項

* 廣告測試：於開發與模擬器測試階段，請務必確認 `AppOpenAdManager` 與 `RewardedAdManager` 呼叫的是測試專用廣告單元 ID，避免因無效曝光受系統懲處。
